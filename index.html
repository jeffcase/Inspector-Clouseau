Inspector-Clouseau by jeffcase

This automation library(.NET) allows you to inspect and modify properties on DOM elements in any browser. Change text input values etc. Works with any browser on any website anywhere in the world. Without having to modify the code on the server.

Sample code List elements = Clouseau.CurrentBrowser.GetElements("#content", propertyNames); string currentValue = elements[0].Properties["html"].CachedValue; elements[0].Properties["html"].ActiveValue = currentValue + "--Added content";
